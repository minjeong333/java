문제5)
1,000개의 수를 읽어 평균을 계산하여 합과 평균을 출력하는 순서도를 작성하시오

0) 1000개의 수 >> 정해진 범위
	1)1000개 입력하기
	2)1000개 랜덤으로 주기
1) 읽어.. >> 범위 안의  행동 >> 반복 / 1000이라는 한계 존재 >> for문 사용
2) 합과 평균 계산 >> 모든 합을 구하고 1000으로 나누기


1. 입력 데이터
    - 변수명: input
    - 자료형: int
    - 설명: 사용자로부터 입력받는 1,000개의 정수

2. 내부 데이터
    - 변수명: limit
    - 자료형: int
    - 값: 1000 (반복 횟수)

    - 변수명: sum
    - 자료형: int
    - 설명: 입력받은 수들의 누적 합계

    - 변수명: i
    - 자료형: int
    - 설명: for문의 반복 제어 변수 (1 ~ 1000)

3. 출력 데이터
    - 변수명: sum,avg
    - 

4. 처리 로직
    0. 초기화: sum = 0 설정
    1. 반복 (for): i가 1부터 1000까지 1씩 증가하며 반복
    2. 입력: 사용자로부터 숫자(input)를 하나씩 읽음
    3. 누적: sum = sum + input 수행
    4. 계산: 반복 종료 후 avg = sum / 1000.0 수행
    5. 출력: sum과 avg 값을 화면에 표시

4. 처리
	0. 데이터 초기화 / int sum=0; int limit=1000;(1000개 다 치기 귀찮으니까 5로 대체해서 사용)
	1. 반복 시작 / for (int i = 1; i <= limit; i++)
		- 1. 사용자 입력받기 / input
		- 2. 총합 더하기 / sum+=input;
	2. 반복 종료
	3. 평균 계산 / double avg = (double) sum / limit;
	4. 결과 출력 및 종료
