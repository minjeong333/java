1.
사용자와 컴퓨터가 가위바위보하는 프로그램을 작성하려고 한다. 
가위,바위,보는 1,2,3의 숫자로 나타냅니다.
컴퓨터는 랜덤으로 손을 선택합니다. 결과화면에는 ‘무승부’, ‘사용자승리’ ,‘사용자패배’ 중 하나를 표시합니다. 무승부의 경우는 다시 승부합니다.
 (승패가 나면 사용자의  승리 또는 패배를 출력한다 )


 - 가위=1 바위=2 보=3 , 결과= '무승부', '사용자승리', '사용자패배'로 설정한다
 - 사용자= 1일때 
     - 컴퓨터= 1 => 결과= '무승부' 
     - 컴퓨터= 2 => 결과= '사용자패배'
     - 컴퓨터= 3 => 결과= '사용자승리'

- 사용자= 2일때(바위)
    - 컴퓨터= 1 => 결과= '사용자승리'
    - 컴퓨터= 2 => 결과= '무승부'
    - 컴퓨터= 3 => 결과= '사용자패배'

- 사용자=3일때(보)
   - 컴퓨터= 1 => 결과= '사용자패배'
   - 컴퓨터= 2 => 결과= '사용자승리'
   - 컴퓨터= 3 => 결과= '무승부'

- 결과가 '무승부'면 과정을 다시 반복한다
- 결과가 '사용자승리' 혹은 '사용자패배'이면 최종결과를 출력한다.

3. 수 맞추기 게임
   컴퓨터가 임의의 수를 내면 사람이 그 수를 맞추는 게임이다 . 맞출 때 까지 반복합니다
   (컴퓨터는 100이내의 임의의 수를 낸다)
   up & down으로  컴퓨터가 낸 수를 맞추기 

 
 (추가옵션사항 : 3번 안에 맞추는 맛있는 간식이 제공된다. ( 간식을 받는 사람에게는 적절한 메시지를 출력한다) 

 - 컴퓨터가 100이내의 임의의 수를 낸다
 - 사용자가 100이내의 임의의 수를 낸다
 - 사용자의 수 > 컴퓨터의 수 => "down" 출력 
 - 사용자의 수 < 컴퓨터의 수 => "up" 출력
 - 사용자의 수 == 컴퓨터의 수까지 과정을 반복한다
 - 사용자가 수를 맞추는 횟수를 3번으로 제한했을 때, 컴퓨터의 수 == 사용자의 수 반복을 멈춤과 동시에 
   "3번 안에 맞춘 당신에게는 맛있는 간식이 제공됩니다"라는 alert창을 낸다.
 


4 .  수맞추기 게임  ver2   
 컴퓨터가 랜덤수를 정합니다. 
 컴퓨터가 랜덤수를 맞추는 게임입니다.
 세 개의 변수가  랜덤수를 맞춥니다. 맞출 때 까지 반복합니다
 세 개의 변수 중에  어떤 변수가  맞췄는지를 사용자가 선택하는 게임입니다.

- 컴퓨터가 랜덤수를 정한다
- 3개의 변수를 도출한다
- 컴퓨터의 랜덤수와 일치하는 변수가 하나가 나올 때까지 과정을 반복한다
- 컴퓨터의 랜덤수와 일치하는 변수가 나왔을 경우 
  사용자가 세 개의 변수 중 어떤 변수가 맞췄는지 선택한다. 
- 사용자가 맞는 변수를 고를 때까지 과정을 반복한다.
- 사용자가 변수를 맞게 골랐으면 최종결과를 도출한다.
- 출력결과는 "맞췄습니다"로 한다
